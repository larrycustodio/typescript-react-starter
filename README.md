## Installation

### 1. Clone the repository on your local drive
```
$ cd <target-repository-dir>
$ git clone extend-api-docs
```

### 2. Install dependencies with [yarn](yarnpkg.com):
```
$ yarn install
```

### 3. Develop
```
$ yarn develop
```
